## 오늘 만드는 것 — 랩 3개

세 랩은 **같은 레포 하나**에 누적됩니다. 끝나면 여러분의 `.bob/`에 파일 네 개가 들어 있고, 그대로 `git add .bob/ AGENTS.md && git commit` 됩니다.

| | 무엇을 만드나 | 어떻게 | 손에 남는 것 |
|---|---|---|---|
| **LAB 1 · Build** | Python 백엔드에 예약 변경 엔드포인트 `POST /bookings/{id}/reschedule` 추가 | `AGENTS.md`에 팀 규칙을 박고 → Ask → Plan → Agent → 테스트 | `AGENTS.md` 팀 컨벤션 + 엔드포인트 + MCP 도구 + 테스트 |
| **LAB 2 · Extend** | 로컬 MCP 서버 붙이기 + 도구 `create_ticket` 직접 제작 + 커스텀 모드로 편집 잠금 | ① 이미 도는 서버 붙여 배선 검증(3분) → ② 도구 1개 구현·빌드·재시작·호출 → ③ 커스텀 모드로 잠금 | `.bob/mcp.json` + MCP 도구 + `.bob/custom_modes.yaml` |
| **LAB 3 · Modernize** | 테스트 0개 레거시 Java 클래스 `QuoteService`에 안전망 씌우기 | `legacy-safety-net` Skill 적용 → 분석 리포트 → 특성화 테스트 생성 → 초록 (버그는 고치지 말고 고정) | Skill + 분석 리포트 + 통과하는 테스트 |

**각 랩의 핵심 장면**
- LAB 1 — 기존 코드는 에러에도 전부 HTTP 200인데 **내 엔드포인트만 404/409** → Bob이 코드가 아니라 내 규칙을 따랐다.
- LAB 2 — Bob이 방금 자기가 짠 도구를 호출해 **`tickets.json`이 실제로 늘어난다.**
- LAB 3 — 리팩터링해도 테스트가 **여전히 초록** = 손대도 안전하다는 확신.

## 오늘의 약속과 심사 기준

### "채팅이 아니라 자산을 내놓는다"

여러분은 "AI가 코드를 짜준다"를 **이미 압니다.** 오늘 배울 것은 그게 아닙니다.
Bob의 차별점은 **레포 안에 살면서, 커밋되고 팀에 공유되는 자산을 만든다**는 것입니다 — `AGENTS.md`, Skill, 커스텀 모드, MCP 서버.

그래서 **문서·리포트 생성도 정당한 과제입니다.** 밸류가 갈리는 지점은 딱 하나 — 파일로 남느냐, 채팅으로 흩어지느냐입니다. 레포를 읽어 매주 구조화된 리포트를 뱉는 Skill은, 생성된 API 엔드포인트와 **동등한 등급의 산출물**입니다.

### 심사 기준은 단 하나

> **"이걸 월요일에 우리 팀 레포에 커밋하겠습니까?"**

이 한 문장이 코드 자산과 문서 자산을 **같은 링 위에 올립니다.**

## 당일 전에 (D-5)

랩은 **여러분 노트북이 당일 오프라인으로 돌아야** 성립합니다. 스타터 zip을 받아 풀고, 자기 사내망에서 한 번 점검하세요.

```bash
bash scripts/preflight.sh   # 스타터 레포 안에 있습니다
```

- 방화벽·프록시, Bob 설치·로그인·Bobcoin, **Advanced 모드**(LAB 3 전제)를 미리 끝냅니다.
- `./mvnw`는 `-o`를 줘도 래퍼 자신이 Maven 배포판을 먼저 받으러 갑니다. preflight가 미리 채웁니다.
- JDK 버전은 신경 쓰지 마세요 — JDK 8·21·23 어디서나 됩니다. 단 `mvn`이 아니라 **`./mvnw`**.
